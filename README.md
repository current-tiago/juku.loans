# juku.loans Website

A modern, responsive website for juku.loans - a fictional lending company. Built with vanilla HTML, CSS, and JavaScript.

## 🚀 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Mobile navigation, smooth scrolling, form validation
- **Accessibility**: Semantic HTML and ARIA labels for better accessibility
- **Performance**: Optimized CSS and JavaScript for fast loading

## 📁 Project Structure

```
juku-loans/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript functionality
└── README.md           # Project documentation
```

## 🎨 Design Features

### Sections
- **Hero Section**: Eye-catching landing area with call-to-action buttons
- **Services**: Showcase of loan offerings (Personal, Home, Auto)
- **About**: Company information and key features
- **Contact**: Contact form and company details
- **Footer**: Links and legal information

### Interactive Elements
- Mobile-friendly hamburger navigation
- Smooth scrolling between sections
- Contact form with validation
- Button hover effects and animations
- Scroll-triggered animations

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **Vanilla JavaScript**: No frameworks, pure JavaScript
- **Google Fonts**: Inter font family
- **CSS Animations**: Smooth transitions and keyframes

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open** `index.html` in your web browser
3. **That's it!** No build process or dependencies required

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px - 1199px
- **Mobile**: Below 768px

## 🔧 Customization

### Colors
The main color scheme can be modified in `css/style.css`:
- Primary Blue: `#2563eb`
- Dark Blue: `#1d4ed8`
- Dark Gray: `#1e293b`
- Light Gray: `#f8fafc`

### Content
- Update company information in `index.html`
- Modify contact details and form fields
- Add or remove service cards as needed

### Styling
- All styles are in `css/style.css`
- Responsive breakpoints are clearly marked
- Animation durations and effects can be adjusted

## 📧 Contact Form

The contact form includes:
- Name, email, and message fields
- Client-side validation
- Success/error notifications
- Form reset after submission

**Note**: This is a frontend-only implementation. For production use, you'll need to add backend functionality to handle form submissions.

## 🎨 Animations

- **Fade-in animations** for service cards and content sections
- **Smooth scrolling** between page sections
- **Button ripple effects** on click
- **Mobile navigation** slide-in/out animations
